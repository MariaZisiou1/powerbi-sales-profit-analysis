# Sales & Profit Analysis – Power BI Project

This project analyzes sales performance, profitability, and discounting strategies using the Sample Superstore dataset.  
It was created as a junior data analyst portfolio project to practice real-world Power BI skills, focusing on clean data analysis, DAX measures, and interactive dashboards.

---

## What's Included
- Sample Superstore dataset with sales, profit, discount, product, category, and regional information
- Interactive Power BI dashboard analyzing:
  - Total sales, total profit, and profit margin
  - Product and category performance
  - Regional performance and trends
  - Discount impact on profitability
- Calculated DAX measures for key metrics
- Clean and readable table and chart visuals
- Slicers for filtering by year, category

---

## Skills Practised

**Data Cleaning & Preparation:**  
- Verified data types and consistency  
- Handled missing values and prepared dataset in Excel before importing to Power BI  

**Data Modeling & DAX:**  
- Used a single clean table for all analysis  
- Calculated measures: Total Sales, Total Profit, Profit Margin (%)  
- Created calculated columns for row-level analysis  

**Data Analysis & Insights:**  
- KPI trends over time  
- Bottom 10 products by profit  
- Discount vs Profit Margin analysis  

**Data Visualization:**  
- Cards, tables, bar charts, scatter plots, maps  
- Clean dashboard layout with professional color palette  
- Interactive slicers for enhanced user experience

---

## Dashboard Pages

### Page 1 – Executive Overview
- High-level KPIs (Sales, Profit, Margin)
- Sales trend over time
- Discount vs Profit Margin scatter plot

### Page 2 – Product Performance
- Profitability by category and sub-category
- Bottom 10 products by profit to highlight loss-making items

### Page 3 – Regional Performance
- Profit analysis by region
- Summary insights

---

## Key Insights
- Sales growth accelerates in later years  
- Higher discount levels are associated with lower profit margins  
- Several products generate negative profit  
- Profitability varies across regions
